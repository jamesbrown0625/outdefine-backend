# Setup Local DB Client in your machine.

If you want to connect remote dev db in your local setup. You need to download any db client. We suggest you [PG Admin](https://www.pgadmin.org/).

Once you have downloaded and install the client in your machine.Open your db client and follow these steps if you have installed pg admin.

1) Create a remote server.
2) Go to connection tab. 
3) Open `db.ts` in backend repo and use the hostname, db , username and password to connect the remote db in your machine.
